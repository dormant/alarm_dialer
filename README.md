# alarm_dialer

## ~/src/alarm_dialer

## alarm_dialer.sh

* Script to dial phones using a USB modem.
* Runs on *opsproc2*.
* Is run over *ssh* by *earthworm* on *winston1*.
* Uses a USB modem. There is a spare somewhere.
* Numbers to be dialled are in *alarm_numbers.txt*.
